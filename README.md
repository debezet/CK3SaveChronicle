# CK3SaveChronicle

Projekt dokumentujący dzieje dynastii Wędrowyczów z kampanii Crusader Kings III.

Głównym dziełem projektu jest **Kronika Domu Wędrowyczów** — stylizowana kronika spisywana głosem Brata Mateusza na podstawie raportów Archiwisty, notatek gracza oraz danych wydobywanych z save'ów CK3.

## Role projektu

- **Archiwista** — wydobywa fakty z save'ów i notatek.
- **Brat Mateusz** — pisze tekst kroniki.
- **Redaktor** — pilnuje spójności, stylu i ciągłości historycznej.
- **Skryba** — składa wydania książkowe.
- **Wydawca** — właściciel repozytorium i osoba publikująca kolejne wersje.

## Struktura

- `chronicle/` — teksty źródłowe kroniki.
- `archive/` — save'y, raporty i notatki źródłowe.
- `docs/` — zasady stylu, decyzje redakcyjne i roadmapa.
- `tools/` — przyszłe narzędzia do analizy save'ów i budowania wydań.
- `output/` — wygenerowane pliki DOCX/PDF, jeśli trzymane lokalnie.

## Status

Repozytorium jest w fazie organizacji źródeł pod **Wydanie III**.
